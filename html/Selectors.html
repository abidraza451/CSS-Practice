Selectors:


selector is used to target the html element so that we can apply the property


Type of Selector:


  1. Element Selector

    The element selector targets all elements of a specific type.

                                
                    /* Element Selector */
                    p {
                        color: blue;
                        font-size: 16px;
                    }

  2. Id Selector

    The Id selector targets an element with a specific ID.

                                
                    /* ID Selector */
                    #unique {
                        background-color: yellow;
                        font-weight: bold;
                    }

  3. Class Selector

    The class selector targets elements with a specific class.

                                
                    /* Class Selector */
                    .className {
                        text-decoration: underline;
                        color: red;
                    }

  4. Universal Selector

    The universal selector targets all elements on the page.

                                
                    /* Universal Selector */
                    * {
                        margin: 0;
                        padding: 0;
                        box-sizing: border-box;
                    }

  5. Grouping Selector

    The grouping selector allows you to apply the same styles to multiple selectors at once.

                                
                    /* Grouping Selector */
                    h1, h2, h3 {
                        color: #333;
                        font-family: Arial, sans-serif;
                    }
