CSS Properties - Color, Font, Text, and Background

  1. Color Properties

    Color properties are used to set the color of text, borders, backgrounds, etc.

    Possible values:

    • Color Name (e.g., red, blue)
    • Hexadecimal color code (e.g., #ff0000)
    • RGB (e.g., rgb(255, 0, 0))
    • RGBA (e.g., rgba(255, 0, 0, 0.5))
    • HSL (e.g., hsl(0, 100%, 50%))
    • HSLA (e.g., hsla(0, 100%, 50%, 0.5))
                            
        /* Color Properties */
        p {
            color: red; /* Color Name */
        }
        
        div {
            background-color: #ff0000; /* Hexadecimal */
            color: rgb(255, 0, 0); /* RGB */
        }

  2. Font Properties

    Font properties control the appearance of text.

    Possible values:

    • font-family (e.g., Arial, sans-serif, 'Times New Roman')
    • font-size (e.g., 16px, 1em, larger)
    • font-weight (e.g., normal, bold, bolder, 100 to 900)
    • font-style (e.g., normal, italic, oblique)
    • font-variant (e.g., normal, small-caps)
    • line-height (e.g., 1.5, 20px)
                            
        /* Font Properties */
        h1 {
            font-family: 'Arial', sans-serif;
            font-size: 24px;
            font-weight: bold;
            font-style: italic;
        }
        
        p {
            line-height: 1.5;
        }

  3. Text Properties

    Text properties control the styling of text, including alignment, decoration, spacing, etc.

    Possible values:

    • text-align (e.g., left, center, right, justify)
    • text-transform (e.g., uppercase, lowercase, capitalize)
    • text-decoration (e.g., underline, line-through, none)
    • letter-spacing (e.g., 1px, 0.1em)
    • word-spacing (e.g., 5px, 0.2em)
    • text-indent (e.g., 20px, 2em)
    • white-space (e.g., normal, nowrap, pre)
                            
        /* Text Properties */
        p {
            text-align: center;
            text-transform: uppercase;
            text-decoration: underline;
        }
        
        h2 {
            letter-spacing: 2px;
            word-spacing: 5px;
            text-indent: 30px;
        }

  4. Background Properties

    Background properties allow you to set background colors, images, and related properties for elements.

    Possible values:

    • background-color (e.g., red, #ff0000, rgba(255, 0, 0, 0.5))
    • background-image (e.g., url('image.jpg'))
    • background-position (e.g., top left, center, 50% 50%)
    • background-size (e.g., cover, contain, 100px 100px)
    • background-repeat (e.g., no-repeat, repeat, repeat-x, repeat-y)
    • background-attachment (e.g., scroll, fixed)
                            
        /* Background Properties */
        div {
            background-color: #ffcc00;
            background-image: url('background.jpg');
            background-position: center;
            background-size: cover;
            background-repeat: no-repeat;
        }
